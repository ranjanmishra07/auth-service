## setup Backend

```bash Run Backend
cd auth-server
cp .env.example .env
modify the .env file to use your MONGO_URI and JWT_SECRET
$ npm install
$ npm start:dev
It will run on port 3001
```

## Setup Frontend

```bash Run Frontend
cd auth-client
$ npm install
$ npm start
It will run on port 3000
```

## Database

```bash
 Run mongodb server
```
